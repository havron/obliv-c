Introduction to Obliv-C

NOTE: This webpage is currently being written and is subject to change frequently.
Obliv-C is a lightweight GCC wrapper that makes it simple for multiple parties to compute a joint function on their private inputs. A full description of the language can be found here. The language comes with numerous built-in protocols which allow a programmer to quickly implement a secure multiparty computation. One particularly useful application of Obliv-C is in allowing for aggregate data analysis from private datasets.

Setting up the Obliv-C Environment

Installation for Ubuntu systems:

  1. Run sudo apt-get install ocaml libgcrypt20-dev.
  2. Clone the repository using git, or download and unpack it from its .zip or .tar.gz file.
  3. Get to the obliv-c directory with cd obliv-c.
  4. Run ./configure && make RELEASE=1.
You are ready to go! Take a look at the code in the tutorial before writing your own programs.

Using Amazon Web Services?

Do you want to start testing your Obliv-C program over servers, but avoid the installation process? Use this (not yet linked) custom Amazon Machine Image (AMI) and quickly launch EC2 instances with an Obliv-C environment ready to go. Run cd ~/obliv-c && git pull before compiling code.

Testing out Obliv-C with data analysis program

You may have noticed a folder in the tutorial directory named 'olinReg'. This is an implementation of linear regression, a tool that allows two parties to compute the correlation of their joint data. See the README to get started. More details will be posted here soon!
