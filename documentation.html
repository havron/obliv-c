Documentation for Obliv-C

About this Documentation

The below documentation of Obliv-C protocols, types, functions, qualifiers, and other aspects of the language is by no means comprehensive, nor is it the standard documentation for the Obliv-C language. The documentation is included for the sole purpose of making it simpler for users to understand how to write programs in Obliv-C successfully, and uses an informal tone to achieve this purpose. The organization of the documentation is alphabetical. Contact the author of the documentation with any issues, corrections, or queries.

Functions

void cleanupProtocol(ProtocolDesc* pd)

Requires #include <obliv.h>. Call this function in your C file after execYaoProtocol() has been called. This function will clean up the ProtocolDesc.

void execYaoProtocol(ProtocolDesc* pd, protocol_run start, void* arg)

Requires #include <obliv.h>. Call this function in your C file's main() function to execute the Yao protocol. Supply the starting function name (without parentheses) in the Obliv-C file as the parameter for protocol_run start. Supply any arguments for the Obliv-C starting function in void* arg.

void setCurrentParty(ProtocolDesc* pd, int party)

Requires #include <obliv.h>. Sets the proper parties into the ProtocolDesc. Call this before calling execYaoProtocol().
